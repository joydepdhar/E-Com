AuraArcade is a modern full-stack e-commerce application built with:

- **Frontend:** React + Vite + TailwindCSS  
- **Backend:** Django REST Framework  
- **Authentication:** JWT (Access + Refresh Tokens)  
- **Hosting:** Render (Backend) + vercel(Frontend)

This project includes product browsing, categories, cart management, user authentication, and a clean UI designed for real-world e-commerce usage.

---

## Features

### Frontend
- Modern UI with React + TailwindCSS
- Product listing and filtering  
- Category-based browsing  
- JWT-protected Add to Cart  
- Responsive Navbar and Hero Section  
- Product detail pages  
- Responsive on all devices  
- Dynamic image loading (Backend/GitHub)

### Backend
- Django REST Framework API  
- JWT Authentication  
- Product, Category, Cart, and User APIs  
- Django Admin support  
- Postgres (Render)
